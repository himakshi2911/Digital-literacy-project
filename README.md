# Digital Literacy Portfolio

#Overview

This repository contains my Digital Literacy Portfolio created as part of my role as a *Student Digital Ambassador*. The project focuses on helping students understand how to use digital tools effectively, communicate professionally, and stay safe online.

##  Objective
The objective of this project is to develop essential digital skills such as:
- Creating digital content
- Building a professional online presence
- Practicing coding and collaboration
- Writing professional emails
- Understanding cyber safety and prevention

##  Project Tasks

###  Task 1: Digital Literacy Awareness Infographic
Created an infographic using Canva to spread awareness about digital literacy and its importance.

###  Task 2: Student Digital Portfolio
Built profiles on platforms like LinkedIn, GitHub, and Kaggle to establish a professional online presence.

### Task 3: Coding & Collaboration Platforms
Explored platforms like HackerRank and GitHub to practice coding and understand collaboration tools.

###  Task 4: Email & Etiquette Guide
Drafted professional emails and created a checklist for responsible social media usage.

### Task 5: Cybercrime Awareness
Developed a case study on cybercrime and created a prevention checklist for staying safe online.
